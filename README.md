# xades

Validate and assemble XAdES baseline signatures.

Built on the [bergshamra](https://github.com/kushaldas/bergshamra) XML security components and [tsp-ltv](https://github.com/kushaldas/tsp-ltv) by Kushal Das.
I have wanted to create a BDOC Rust library for a while now, and thanks to his libraries, this was finally possible.
